  1  # -*- coding: utf-8 -*- 
  2  """PySourceInfo - runtime type information on Python source location: package, module, and caller. 
  3   
  4  This modules provides for the location of Python execution 
  5  by means of the package 'inspect' extended by additional sources 
  6  for a simple API. 
  7   
  8  The stack frame of inspect is in particular reduced to the common 
  9  parameter 'spos', which is an abstraction of the 'stack-position' 
 10  representing the level of history within the caller level. 
 11  The value 'spos==0' is the function itself, whereas 'spos==1' is the 
 12  first level caller. Consequently 'spos==2' is the caller of the caller, 
 13  etc. 
 14   
 15  The categories of provided RTTI comprise: 
 16   
 17  * **packages** - Python packages. 
 18   
 19  * **modules** - Python modules - a.k.a. source files. 
 20   
 21  * **callers** - Python functions and class/object methods. 
 22   
 23   
 24  Where the following attributes are available: 
 25   
 26  * name(package, module, function) 
 27   
 28  * OID - dotted relative path to matching item of sys.path 
 29   
 30  * filename 
 31   
 32  * filepathname 
 33   
 34  * item of sys.path 
 35   
 36  * relative path to item of sys.path 
 37   
 38  * line number 
 39   
 40  Dependent on the call context, some of the attribute values may 
 41  not be available. 
 42  E.g. when called from within the python/ipython shell, or 'main'. 
 43   
 44  The API is designed here as a collection of slim functions 
 45  only in order to avoid any overhead for generic application. 
 46   
 47  """
