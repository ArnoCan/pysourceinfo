testdata Package¶

Common test data for Use-Cases¶

Common data for ‘UseCases’ and ‘tests’. Refer to the package by PYTHONPATH. The global variable ‘testdata.mypath’ provides the pathname into ‘testdata’.

The following directory tree is provided:
